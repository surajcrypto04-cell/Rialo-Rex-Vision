# Rialo REX-Vision Frontend

React dashboard that visualizes Rialo's architecture. Connects to the backend simulator via WebSocket.

## Features

- **SfS Gauge** - shows credit balance refilling from staking rewards
- **Concurrency Toggle** - switch between optimistic/pessimistic modes
- **Event Stream** - live Web2 API events with 0ms latency
- **Transaction Canvas** - real-time tx processing visualization
- **Use Case Simulator** - interactive demos for different scenarios
- **Comparison Panel** - Rialo vs traditional blockchain comparison
